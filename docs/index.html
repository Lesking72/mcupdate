Minecraft 1.15.2

Next up, Nether!

+Bees no longer anger when a nearby nest/hive is broken with a silk touch tool
+Added doPatrolSpawning and doTraderSpawning gamerules
+Added gui_light option in block models to allow controlling light when rendering model as item in GUI
+Any birch or oak sapling grown near a flower within 2 blocks distance on the same y-level has a 5% chance of having a bee nest
+Bee nests now have a 2% chance of spawning in Flower Forests
+Bee nests now have a 0.2% chance of spawning in Forest, Wooded Hills, Birch Forest, Tall Birch Forest, Birch Forest Hills, and Tall Birch Hills biomes
+Fixed bugs

Minecraft 1.14.4

+Fixed Bugs
+Improved Performance
+Suspicious stew made from poppies now gives you night vision instead of speed
+Added "/debug report" for getting more detailed bug reports
+Fixed a memory leak
+Removed camera pivot offset in first person
+Improved chunk loading when traveling at high speed
+Fixed incorrect Pillager texture
+/reload and /forceload are now available to gamemasters
+function-permission-level is a new setting in server.properties that controls which commands functions have access to
+Villagers now stock more items
+Villagers now remember their gossip after becoming a zombie villager
+Gossip about players who converted a zombie villager now lasts longer
+The Player Activity button has been removed from the Realms screen
+Villagers now wait to restock until they have trades that need restocking
+Sieges no longer occur on mushroom islands
+Mobs will no longer attempt to pathfind through bamboo
+Pinged the human

Minecraft 1.14.3

+Items can be repaired by crafting them together again
+Reverted the enchanting system to how it was before 1.14
+Torches, lanterns and pressure plates can now be placed on glass panes and iron bars
+Saturation is no longer required for the "How did we get here?" advancement
+You can now turn off raids with '/gamerule disableRaids true'
+Vindicators are no longer part of patrols
+Doubled the minimum time to spawn patrols from ~5 minutes to ~10 minutes
+Patrols no longer spawn if the block light level disallows monster spawning
+Patrols now spawn in any biome except mushroom biomes
+Villagers that are panicking now have a higher chance of spawning Iron Golems, assuming they can occasionally work and sleep
+"Last slept" and "last worked" is now saved properly for villagers
+Farmers now spend more time farming when they are working
+Farmers can now always give away food, even if other villagers don't need it
+Improved performance
+Fixed bugs

Minecraft 1.14.2

We've fixed some more issues left in 1.14.1. We wanted to get these fixes out as soon as possible as they affect gameplay, but we also have a bunch of less critical fixes coming in 1.14.3

+All light will now be re-calculated the first time a world from a previous version is loaded
+Errors encountered while loading regions now print more diagnostics to the log file
+Fixed a bug where parrots spawn on grass, not grass blocks
+Added server-side chunk count to debug screen
+Modified raider spawning so they only spawn in fully loaded chunks
+Increased search radius for bell when starting a raid from 48 to 64 blocks
